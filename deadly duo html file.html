Application features

Four core Python functions powering the Parks Explorer

display_all_parks()
View all parks
Shows all 8 parks with name, location, description and full list of activities in a clean formatted layout.
search_parks(keyword)
Search by keyword
Searches across park name, location and description simultaneously. Fully case-insensitive. Partial matches work.
filter_by_activity(activity)
Filter by activity
Returns only parks that offer the chosen activity. Handles case differences automatically. Returns empty list if no match.
get_all_activities()
View all activities
Returns a complete, alphabetically sorted, deduplicated list of all activities across the entire park database.
